Pernyataan Aksesibilitas

Diperbarui: April 2026

♿ Komitmen Kami

GasLamar berkomitmen untuk memastikan aksesibilitas digital bagi semua orang, termasuk penyandang disabilitas. Kami terus berupaya meningkatkan pengalaman pengguna dan menerapkan standar aksesibilitas yang relevan.

Target kami adalah memenuhi WCAG 2.1 Level AA (Web Content Accessibility Guidelines versi 2.1, tingkat kepatuhan AA) di seluruh antarmuka GasLamar.

📊 Status Kepatuhan

GasLamar sebagian memenuhi WCAG 2.1 Level AA. Kami secara aktif mengidentifikasi dan memperbaiki area yang belum sepenuhnya memenuhi standar ini.

Langkah-langkah yang telah diterapkan antara lain:

  • Skip-to-content link di setiap halaman utama untuk pengguna keyboard
  • Ukuran touch target minimal 44×44 px untuk semua elemen interaktif
  • Label aria-label dan aria-describedby pada form dan elemen dinamis
  • Rasio kontras warna yang memenuhi WCAG AA (minimal 4.5:1 untuk teks normal)
  • Struktur heading yang hierarkis dan semantik HTML yang benar
  • Atribut alt pada semua gambar bermakna
  • Navigasi keyboard penuh tanpa memerlukan mouse

⚠️ Keterbatasan yang Diketahui

Area berikut masih dalam proses perbaikan:

  • Beberapa komponen animasi mungkin belum sepenuhnya mengikuti preferensi prefers-reduced-motion
  • Dokumen CV yang dihasilkan (PDF/DOCX) belum dioptimalkan untuk screen reader — ini merupakan keterbatasan format file, bukan antarmuka web

Jika kamu menemukan hambatan aksesibilitas lain, harap beritahu kami melalui kontak di bawah.

🛠️ Pendekatan Teknis

Upaya aksesibilitas kami meliputi:

  • Pengujian manual dengan keyboard-only navigation secara berkala
  • Penggunaan React dengan atribut ARIA yang tepat pada halaman dinamis
  • HTML semantik (<main>, <nav>, <header>, <footer>) di semua halaman
  • Form dengan label yang benar dan pesan error yang deskriptif
  • Focus management yang terlihat jelas pada semua elemen interaktif

📬 Umpan Balik Aksesibilitas

Kami menyambut masukan terkait aksesibilitas dari semua pengguna. Jika kamu menemukan hambatan atau memiliki saran untuk meningkatkan aksesibilitas GasLamar, hubungi kami:

Email: support@gaslamar.com

Sebutkan halaman yang terpengaruh, teknologi assistif yang kamu gunakan (misalnya screen reader, switch control), dan deskripsi hambatan yang dialami. Kami berkomitmen untuk merespons dalam 5 hari kerja.

Accessibility Statement

Updated: April 2026

♿ Our Commitment

GasLamar is committed to ensuring digital accessibility for people with disabilities. We continually improve the user experience and apply relevant accessibility standards.

Our target is conformance with WCAG 2.1 Level AA (Web Content Accessibility Guidelines version 2.1, conformance level AA) across all GasLamar interfaces.

📊 Conformance Status

GasLamar partially conforms to WCAG 2.1 Level AA. We are actively identifying and addressing areas that do not yet fully meet this standard.

Measures already in place include:

  • Skip-to-content link on every main page for keyboard users
  • Minimum 44×44 px touch targets for all interactive elements
  • Appropriate aria-label and aria-describedby attributes on forms and dynamic elements
  • Colour contrast ratios meeting WCAG AA (minimum 4.5:1 for normal text)
  • Hierarchical heading structure and correct semantic HTML
  • alt attributes on all meaningful images
  • Full keyboard navigation without requiring a mouse

⚠️ Known Limitations

The following areas are still being improved:

  • Some animated components may not yet fully respect the prefers-reduced-motion media query
  • Generated CV documents (PDF/DOCX) are not screen-reader optimised — this is a limitation of the file format, not the web interface

If you encounter any other accessibility barriers, please let us know using the contact below.

📬 Accessibility Feedback

We welcome accessibility feedback from all users. If you find a barrier or have suggestions to improve GasLamar's accessibility, please contact us:

Email: support@gaslamar.com

Please include the page affected, the assistive technology you are using (e.g. screen reader, switch control), and a description of the barrier. We aim to respond within 5 business days.
